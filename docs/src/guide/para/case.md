## Parameters

### case

---

**Name**: case

**Type**: String

**Details**: Short name of the system that you would like to study with `Zen`. Note that underscores and blank spaces are not permitted. 

**Syntax**: 

```toml
case = "seedname"
```

**Examples**:

```toml
case = "SrVO3"
case = "NiO"
case = "Ce115"
```
